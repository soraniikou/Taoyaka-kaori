Taoyaka-kaori 🌸


![アプリ体験→]https://taoyaka-kaori.vercel.app



たおやかな香りを纏うような、光るアート体験

p5.jsで制作したインタラクティブなビジュアルアート作品です。  
青く柔らかく光る花のような粒子が、たおやかに揺れます。

使い方

ブラウザで開くと、アートが自動的に動き始めます。

 技術

- p5.js
- HTML / CSS / JavaScript
